#### Run
```php app.php -h```

#### Requirement
  
- At least PHP 5.3  
- PHP cURL Extension  
- PHP JSON Extension  


#### Install

*Installing on Windows*  

```Download http://windows.php.net/downloads/releases/php-5.3.21-Win32-VC9-x86.msi```  

```Tick cURL Extension on installation step```
